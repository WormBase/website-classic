Structure Data Status Tags

Structure information is imported from TargetDB and PDB. For records imported from TargetDB, the following tags(*) are used to indicate the status of the targets in the structure determination pipeline, as provided by individual genomic centers. A status tag of "In PDB" has been added to records extracted from PDB.

* This list has been extracted from TargetDB documentation.

  • Cloned
  • Expressed
  • Soluble
  • Purified
  • Crystallized
